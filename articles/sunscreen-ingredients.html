紫外線の種類 — UVA・UVB・UVCの違い

日焼け止めを正しく選ぶためには、まず「何から肌を守るのか」を知る必要があります。太陽光に含まれる紫外線は波長によってUVA（315-400nm）、UVB（280-315nm）、UVC（100-280nm）の3種類に分類されます。

UVCはオゾン層でほぼ完全に吸収されるため、地表には届きません。日焼け止めで対策すべきなのはUVAとUVBの2つです。

UVBはエネルギーが強く、肌表面の表皮に作用してサンバーン（赤くなる日焼け）を引き起こします。シミの原因となるメラニン生成のトリガーにもなり、短時間でもダメージが蓄積します。ただし、ガラスや雲である程度遮断されるため、屋内にいれば影響は限定的です。

一方、UVAは波長が長く、真皮層まで到達します。コラーゲンやエラスチンを破壊し、シワやたるみといった「光老化」の主因となります。UVAはガラスも透過し、曇りの日でも地表に届く量はほとんど変わりません。つまり、エイジングケアの観点ではUVA対策こそが重要なのです。

Key Point

UVBは「表皮のダメージ（赤み・シミ）」、UVAは「真皮のダメージ（シワ・たるみ）」。日焼け止めを選ぶ際は、両方への防御力をチェックする必要があります。

SPFとPA表示の正しい読み方

ドラッグストアで「SPF50+ PA++++」という表示を見て、「数値が高いほうが良い」と漠然と選んでいませんか？　SPFとPAは測定対象がそもそも異なります。

SPF（Sun Protection Factor）はUVBに対する防御指標です。「素肌の場合と比べて、サンバーンが起きるまでの時間を何倍に延ばせるか」を示します。たとえばSPF30なら、何も塗らない場合に比べて約30倍の時間、UVBによる赤みを防げるという意味です。ただし、これは規定量（1cm²あたり2mg）を均一に塗った場合の数値であり、実際の使用では塗りムラや量不足により効果は低下します。

PA（Protection Grade of UVA）はUVAに対する防御指標で、日本独自の表記です。PA+からPA++++まで4段階あり、+が多いほどUVA防御力が高いことを意味します。日常使いならPA++以上、炎天下のレジャーならPA++++が推奨されます。

Note

SPF30はUVBの約96.7%を、SPF50は約98%をカットします。差はわずか1.3%。数値を上げるほど肌への負担が増す製品もあるため、日常使いではSPF30～35・PA+++程度で十分なケースが多いです。「高ければ安心」ではなく、生活シーンに合った数値を選びましょう。

紫外線吸収剤とは — 仕組みとメリット・デメリット

紫外線吸収剤は、紫外線のエネルギーを化学的に吸収し、熱などの別のエネルギーに変換して放出する有機化合物です。代表的な成分には以下のようなものがあります。

メトキシケイヒ酸エチルヘキシル（表示名：エチルヘキシルメトキシシンナメート）は、世界で最も広く使われるUVB吸収剤です。使用感が軽く、多くの日焼け止めのベース成分になっています。オキシベンゾン-3はUVAとUVBの両方を吸収できる成分ですが、アレルギー反応のリスクやサンゴ礁への環境影響が指摘され、ハワイ州では2021年からサンゴに有害な日焼け止め成分の販売が規制されています。そのほか、ジエチルアミノヒドロキシベンゾイル安息香酸ヘキシル（UVA吸収剤）、ビスエチルヘキシルオキシフェノールメトキシフェニルトリアジン（広域UV吸収剤）なども成分表示でよく見かけます。

メリットとしては、透明で白浮きしないこと、テクスチャーが軽くのびが良いこと、高いSPF値を実現しやすいことが挙げられます。化粧下地やトーンアップ系の日焼け止めに多く採用されるのはこのためです。

デメリットは、紫外線を吸収する化学反応の過程で成分自体が徐々に分解されるため、効果が時間とともに低下する点です。こまめな塗り直しが不可欠です。また、まれに接触性皮膚炎を起こす人がおり、敏感肌には合わない場合があります。

紫外線散乱剤とは — 仕組みとメリット・デメリット

紫外線散乱剤は、肌の表面に微粒子の膜をつくり、紫外線を物理的に反射・散乱させるアプローチです。主に使われるのは酸化亜鉛（Zinc Oxide）と酸化チタン（Titanium Dioxide）の2つです。

酸化亜鉛はUVAに対する防御力が高く、抗炎症作用もあるため敏感肌向け処方に重用されます。酸化チタンはUVBの散乱効率が特に高く、両者を組み合わせることでUVA・UVBの広い波長域をカバーできます。

メリットは、化学反応を伴わないため成分の分解がほとんど起きず、効果の持続性が高い点です。肌への刺激も少なく、敏感肌用やベビー用日焼け止めの多くが散乱剤ベースで設計されています。

デメリットとしてよく挙がるのが「白浮き」と「きしみ」です。粒子が光を反射するため、塗った箇所が白く見えやすい。また、粉体特有のきしみ感があり、使用感では吸収剤に劣ると感じる人が多いのが実情です。近年はナノ化技術で粒子を微細化し、白浮きを軽減した製品も増えていますが、ナノ粒子の安全性についてはまだ議論が続いています。

Note

まれに酸化亜鉛で肌荒れを起こす方がいます。金属アレルギーの自覚がある場合は、酸化チタンのみ使用の製品を選ぶか、パッチテストをしてから使い始めましょう。

ハイブリッド処方という選択肢

近年のトレンドは、吸収剤と散乱剤を組み合わせたハイブリッド処方です。吸収剤の「使用感の良さ・高SPF」と、散乱剤の「低刺激・持続性」を両立させることを目指した設計です。

たとえば、ベースに酸化亜鉛と酸化チタンを配合しつつ、吸収剤を少量加えてSPF50+を実現する製品や、吸収剤をカプセルに包み込んで肌への直接接触を減らす「カプセルインUV」技術を採用した製品もあります。アネッサやアリィーといった国内の人気ブランドの多くがハイブリッド処方を採用しています。

ハイブリッド処方の利点は、単独処方よりも各成分の配合量を抑えられるため、肌負担と使用感のバランスを取りやすいことです。ただし「ハイブリッド＝万能」ではなく、どの成分がどの程度配合されているかを成分表示で確認することが大切です。

成分表示の読み方 — 覚えるべき成分名

日焼け止めの成分表示を読む際、まず確認したいのは「吸収剤ベースか、散乱剤ベースか、ハイブリッドか」という処方タイプです。次の成分名を覚えておけば、パッケージの裏面から処方の方向性を読み取れます。

紫外線吸収剤の代表的な成分名：メトキシケイヒ酸エチルヘキシル、オキシベンゾン-3、ジエチルアミノヒドロキシベンゾイル安息香酸ヘキシル、ビスエチルヘキシルオキシフェノールメトキシフェニルトリアジン、t-ブチルメトキシジベンゾイルメタン、オクトクリレン。

紫外線散乱剤の代表的な成分名：酸化亜鉛、酸化チタン。この2つだけなので非常にシンプルです。

化粧品の全成分表示は、配合量の多い順に記載されるルールです（医薬部外品は順不同の場合あり）。成分リストの上位に酸化亜鉛・酸化チタンがあれば散乱剤メインの処方、メトキシケイヒ酸エチルヘキシルなどの吸収剤名が上位にあれば吸収剤メインの処方と判断できます。

Tip

店頭で成分名を暗記する必要はありません。「酸化亜鉛」「酸化チタン」の2語だけ覚えておけば、散乱剤が入っているかどうかはすぐ分かります。それ以外のカタカナの長い成分名が並んでいたら、吸収剤ベースの可能性が高い、という読み方で十分です。

まとめ — 肌質別の選び方ガイド

日焼け止め選びは「SPFの数値」だけで完結しません。成分表示から処方タイプを読み取り、自分の肌質と生活スタイルに合った一本を見つけることが、長期的な肌の健康につながります。

Skin Type Guide

敏感肌・乾燥肌 → 散乱剤ベース（酸化亜鉛+酸化チタン）の製品が第一選択。吸収剤フリーの「ノンケミカル」表記を目印に。SPF30・PA+++程度で日常使いに十分。
脂性肌・混合肌 → ハイブリッド処方が使いやすい。吸収剤の軽い使用感で化粧崩れもしにくく、散乱剤の持続性も得られる。
普通肌・健康な肌 → 吸収剤ベースでもハイブリッドでもOK。使用感の好みで選んで問題なし。ただし2〜3時間おきの塗り直しは必須。
アウトドア・レジャー → SPF50+・PA++++のウォータープルーフ。汗・水に強いハイブリッド処方が最適。帰宅後のクレンジングも忘れずに。

どの処方タイプを選んでも、最も大切なのは「適量を塗り、こまめに塗り直す」こと。高スペックの日焼け止めでも、塗る量が足りなければ表示通りの効果は発揮されません。顔全体でおよそパール粒2個分が目安です。

成分表示を読む力は、一度身につければ一生使えるスキルです。次にドラッグストアで日焼け止めを手に取ったとき、ぜひパッケージの裏面を確認してみてください。「何が入っているか」が分かるだけで、日焼け止め選びの精度は格段に上がります。
